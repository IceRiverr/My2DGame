彩虹打砖块小游戏

---------------------------------------
# 游戏定义
* 传统的打砖块游戏

---------------------------------------
# 需求定义：

* 可被击碎的砖块，先只实现一种砖块
* 只可左右移动的球拍
* 一只不断碰撞的乒乓球

---------------------------------------
# GameRule
	游戏结束条件，打光场景中所有的砖块，win
	游戏最大失误次数，都被用光，则失败；
	失误判断是，乒乓球从场景的底部出去，则代表失误次数减少1；

---------------------------------------
# 扩展功能
* 添加背景图片
* 游戏的初始化界面，退出界面，关卡选择界面；
* 多加几种砖块，添加砖块的寿命；
* 砖块被击碎时，可以播放动画
* 给游戏添加墙壁
* 实现关卡加载，切换，至少制作两关；
* 有一个日本的打砖块游戏，给这种游戏添加了道具，类如提高乒乓球运动速度的道具，炸弹道具，减速道具，扩大道具等；

---------------------------------------
# 技术难点
* UI界面，整体还没有框架
* 碰撞检测，完全没有
